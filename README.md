# Gravador de áudio

Gravador de áudio simples utilizando a library [Recorder.js](https://github.com/mattdiamond/Recorderjs).

Recursos:

- Gravador de áudio;
- Pausar gravação;
- Finalizar gravação;
- Opção de salvar no disco;
- Opção de enviar para o servidor;
